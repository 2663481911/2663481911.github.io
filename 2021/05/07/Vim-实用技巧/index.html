发表于 2021-05-07 更新于 2021-09-29

Vim-实用技巧

大小写

命令 用途
g~ 反转大小写
gU 转换为大写
gu 转换为小写
guu 转换当前行小写
gUU 转换当前行为大写

注释

  • gcc 注释当前行
  • gcap 注释当前段
  • gcG 注释当前行到结尾的内容

删除

  • 插入模式下
命令 用途
<C - h> 删除前一个字符（同退格键）
<C - w> 删除前一个单词
<C - u> 删除到行首

插入模式返回普通模式

命令 用途
esc 返回普通模式
<C - [> 返回普通模式
<C - o> 返回普通模式执行一次命令后回到插入模式

替换

命令 用途
u 替换当前字符（后面跟替换的字符）
U 进入替换模式

选择

  • gv 选择上次可视模式选择的文本范围
JavaScript-位运算符
算法-树-深度优先遍历
