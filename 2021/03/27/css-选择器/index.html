后代选择器

后代选择器用于选取某元素的后代元素。

  • 父元素和后代元素用空格隔开
  • 最终选择的是后代元素
  • 后代元素可以是儿子也可以是孙子，只要是父元素的后代就可以
  • 父元素和后代元素可以是任意的基础选择器

子选择器

与后代选择器相比，子元素选择器只能选择作为某元素子元素的元素。

语法：父元素 > 子元素 { 样式声明 }

  • 只能选择div的第一层a元素，不能选择div下li里面的a

  • 最终选择的是子元素

相邻兄弟选择器

选择在第一个元素之后相邻的元素，二者有相同父元素。

后续兄弟选择器

选取所有指定元素之后的兄弟元素。

并集选择器

选择多个元素

语法：元素1, 元素2 { 样式 }
